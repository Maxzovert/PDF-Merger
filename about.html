About the PDF-X

This is my first project after I learned complete JavaScript. So as a beginner i just made it with limited capacity of mergeing only two pdfs into one pdf. It is a PDF Merge Tool name PDF-X allows users to merge two PDF files into a single PDF document. Built with JavaScript, HTML, CSS, Express.js, and MS PowerShell, it offers a simple and efficient way to combine your PDF files.
